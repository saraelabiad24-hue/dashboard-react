# Dashboard-React

## Project Overview
This project is a simple **user management dashboard** built with React JS.  
It allows users to perform basic CRUD operations (Create, Read, Update, Delete).

The goal of this project is to demonstrate skills in React development, component structuring, and state management using hooks.

## Features

Add new users (name & email)  
Display list of users dynamically  
Edit existing user information  
Delete users from the list  
Real-time UI updates using React state 

## Technologies Used

React JS
JavaScript (ES6+)
HTML5
CSS3
React Hooks (useState, useEffect)
